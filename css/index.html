Cascading Style Sheets

La hoja de estilo en cascada o CSS (siglas en inglés de cascading style sheets)
es un lenguaje usado para definir la presentación de un documento estructurado escrito en HTML y derivados.
El W3C es el encargado de formular la especificación de las hojas de estilo que servirán de estándar para los agentes de usuario o navegadores.


Existen tres formas de añadir estilos con css

  1. En una etiqueta: con la el atributo style
  2. En el encabezado: con la etiqueta <style>
  3. En un archivo externo.

Sintaxis de una instruccion en css 

            h1{
                color:blue;
                font-size: 12px;
            }
